PR request

Na het afwerken van je codeblock, of feature die je wou oplossen, ishet de bedoeling je branch af te sluiten, en dus terug in de mainbranch te mergen. Indien je in team werkt, kan het opportuun zijn tewerken met Pull requests. Dit zijn “aanvragen” die je open stelt naarandere developers van een nieuwe toevoeging aan de code base, met devraag dit na te kijken. Tijdens dit proces wordt de code dusonderworpen aan een quality check, en aan eventueel vergeten features.

Dit is een einde.
